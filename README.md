# blank_present

[![Build Status](https://travis-ci.org/foo9/blank_present.svg?branch=master)](https://travis-ci.org/foo9/blank_present)

The blank and present function for PHP.

## Requirements

PHP 5.4+

## Usage

The blank function:

``` php
if (_blank($var)) {
    // do stuff
}
```

The present function:

``` php
if (_present($var)) {
    // do stuff
}
```

## Running tests

``` bash
$ vendor/bin/phpunit
```
